# 🚀 FoundrAI

> **The AI Operating System for Startup Founders**

FoundrAI is an AI-powered startup platform that helps founders transform an idea into a validated business using a collaborative team of AI agents.

Instead of using a single chatbot, FoundrAI coordinates multiple specialized AI agents to analyze ideas, perform market research, generate business strategies, and help founders build startups faster.

---

## ✨ Features

* 🤖 Multi-Agent AI Collaboration
* 💡 AI Startup Idea Generator
* 📊 Startup Dashboard

# 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/muneesharavala/FoundrAI.git
```

Open the project:

```bash
cd FoundrAI
```

Frontend:

```bash
cd apps/web
npm install
npm run dev
```

Backend:

```bash
cd apps/api
pip install -r requirements/base.txt
uvicorn app.main:app --reload
```
