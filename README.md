# FoundrAI

FoundrAI is a monorepo for the platform's web application, API, background services, shared packages, infrastructure, and documentation.

## Repository Structure

- apps/api
- apps/web
- packages/ui
- packages/config
- packages/shared
- packages/types
- services/ai-engine
- services/analytics
- services/notifications
- services/search
- infrastructure
- docs
- scripts
- .github

## Getting Started

1. Review the service-specific directories.
2. Configure environment files from the provided `.env.example` templates.
3. Run the stack with Docker Compose.
